# Self-Configuring RAG Hyperparameter Optimizer

## Overview

This project implements a Self-Configuring Retrieval-Augmented Generation (RAG) System that automatically identifies the best RAG configuration using Optuna Hyperparameter Optimization and Semantic Similarity Evaluation.

The system allows users to upload a PDF document, automatically evaluate multiple RAG configurations, select the best-performing parameters, and answer questions using the optimized configuration.

---

## Problem Statement

Traditional RAG systems require manual tuning of retrieval parameters such as:

- Chunk Size
- Chunk Overlap
- Top-K Retrieval

Finding the optimal configuration often requires multiple experiments and manual evaluation.

This project automates the process by:

1. Testing multiple hyperparameter combinations.
2. Evaluating answer quality using semantic similarity.
3. Selecting the best-performing configuration automatically.

---

## Features

### PDF Upload
Users can upload PDF documents through a Streamlit interface.

### Automatic Hyperparameter Optimization
The system automatically optimizes:

- Chunk Size
- Chunk Overlap
- Top-K Retrieval

using Optuna.

### Semantic Similarity Evaluation
Generated answers are compared against predefined ground-truth answers using:

- Sentence Transformers
- Cosine Similarity

to determine answer quality.

### Vector Search
Document chunks are converted into embeddings and stored in a FAISS vector database.

### Question Answering
Users can ask questions related to the uploaded document and receive answers generated by the optimized RAG pipeline.

### Interactive Streamlit Interface
Provides a user-friendly interface for document upload, optimization, and question answering.

## Installation

### Clone Repository

```bash
git clone https://github.com/saiiswarya02/self-configuring-rag-optimizer.git

cd self-configuring-rag-optimizer
```

### Create Conda Environment

```bash
conda create -n selfrag python=3.11 -y

conda activate selfrag
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

## Workflow

### Step 1
Upload a PDF document.

### Step 2
Click **Optimize RAG**.

### Step 3
Optuna evaluates multiple configurations:

- Chunk Size
- Chunk Overlap
- Top-K Retrieval

### Step 4
Answers are compared with ground-truth answers using semantic similarity.

### Step 5
The best-performing configuration is selected.

### Step 6
Ask questions and receive answers from the optimized RAG system.
